Snake Game
This is a remake of the classic snake game. Totaling 245 lines of code, this project tought me how to create interactive elements, which are used in all of my web designs.
  • manipulating and updating a GUI window
  • keeping track of a player's score
  • accounting for collisions with the sides of the window
  • spawning food at random places on the gameboard
  • ability to change game settings, such as the speed of the snake
